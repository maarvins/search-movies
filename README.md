## 🚀 Getting started

Clone the project and access the folder.

```bash
$ git clone https://github.com/maarvins/search-movies.git && cd search-movies
```

Follow the steps below

```bash
# Install the dependencies
$ npm install


# Start the project
$ npm start
```
